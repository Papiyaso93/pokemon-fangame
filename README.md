# Pokémon Fangame

Fan-game Pokémon développé avec Godot 4.

## Setup

1. Installer [Godot 4](https://godotengine.org/download/)
2. Cloner le repo
3. Ouvrir le projet dans Godot (`project.godot`)
4. Lancer avec le bouton Play ▶

## Stack

- **Moteur** : Godot 4
- **Langage** : GDScript
